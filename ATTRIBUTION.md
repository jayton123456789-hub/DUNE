# Attribution and asset provenance

## Shipped assets

Driftline's shipped visual and audio assets are original to this repository:

- `assets/ball.svg` — centered Aqua ball export with a rotationally balanced current mark.
- `assets/coin.svg` — honey-gold Driftline coin and horizon insignia.
- `assets/icon.svg` — mask-safe Sunwake app icon composition.
- `assets/icon-192.png`, `assets/icon-512.png`, and `assets/apple-touch-icon.png` — raster exports of the authored app icon for install surfaces.
- `assets/background.svg` — wide Sunwake paper-cut desert composition.
- `src/art.js` — procedural ball layers, skin decals, coin rendering, and world preview art.
- `src/sand-renderer.js` and `src/sand-effects.js` — procedural scenery, trails, particles, and gameplay effects.
- `src/game-ui.js` — Web Audio synthesis for interface feedback, wind, launches, landings, rewards, and collisions.
- Inline interface icons in `index.html` — original SVG paths authored for this game.

These files do not incorporate stock art, external icon sets, audio samples, web fonts, or generated copies of another game's assets. No third-party attribution is required for the current shipped bundle.

## Reference disclosure

The control premise was independently recreated as a tribute to the mobile game *Dune!* by Voodoo. No code, artwork, audio, branding, level data, or proprietary assets from that title are included. Driftline is an independent, unofficial project and is not affiliated with or endorsed by Voodoo.

Public UI and game-feel examples were reviewed only as design references during development. They are not redistributed by this project.
